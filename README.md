- 📡 Monitor multiple IP addresses
- ✅ Detect device status (Active / Down)
- ⚡ Measure network latency
- 🔄 Auto-refresh dashboard (every 5 seconds)
- ➕ Add new IP addresses dynamically
- 📊 Visualize latency using graphs
- 📝 Logging system for monitoring history
- ⚠️ Alerts for down devices
